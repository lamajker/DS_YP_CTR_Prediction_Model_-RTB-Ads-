# DS_YP_CTR_Prediction_Model_-RTB-Ads-

Модель машинного обучения для прогнозирования вероятности клика (CTR - Click-Through Rate) в RTB-аукционе.

## О проекте

В проекте реализована модель бинарной классификации, предсказывающая вероятность клика по рекламному объявлению.

Задача имеет прикладную ценность и используется для:

- оптимизации ставок в реальном времени;
- повышения эффективности рекламных кампаний;
- увеличения дохода от показов;
- более точного таргетинга пользователей.

Классы:

- 1 - пользователь кликнул;
- 0 - пользователь не кликнул.

## Цель

Построить модель, которая:

- корректно ранжирует объекты по вероятности клика;
- выдаёт адекватные вероятностные оценки;
- устойчива к дисбалансу классов;
- пригодна для использования в продакшене.

## Данные

Используется датасет рекламных показов:

- признаки пользователя;
- параметры аукциона;
- характеристики площадки;
- категориальные и числовые признаки;
- машинно-сгенерированные признаки (ml_feature_*).

Особенность данных - выраженный дисбаланс классов.

## Используемый стек

- Python
- pandas, numpy
- scikit-learn
- matplotlib
- joblib

## Pipeline модели

### Предобработка

- заполнение пропусков;
- масштабирование числовых признаков;
- кодирование категориальных признаков:
  - One-Hot Encoding;
  - Target Encoding.

### Отбор признаков

- удаление коррелирующих признаков;
- удаление низковариативных признаков;
- RFECV.

### Модель

- LogisticRegression.

### Калибровка

- Isotonic Regression;
- отдельная калибровочная выборка (без утечки данных).

## Метрики

Основная:

- PR-AUC.

Дополнительные:

- Brier score;
- ECE;
- MCE;
- Precision, Recall, F1-score.

## Результаты

### Базовая модель

| Модель            | PR-AUC | Brier |
|-------------------|--------|-------|
| DummyClassifier   | 0.172  | 0.142 |

### Финальные модели

| Модель                     | PR-AUC | Brier | ECE   |
|---------------------------|--------|-------|-------|
| LogisticRegression        | 0.370  | 0.210 | 0.272 |
| LogisticRegression (iso)  | 0.352  | 0.128 | 0.008 |

## Интерпретация результатов

- модель более чем в 2 раза лучше baseline по PR-AUC;
- калибровка улучшает качество вероятностей;
- наблюдается компромисс:

  - PR-AUC снижается;
  - Brier и ECE улучшаются.

Важно:

- PR-AUC отражает качество ранжирования;
- Brier и ECE отражают качество вероятностей.

## Важность признаков

Наибольшее влияние оказывают:

- параметры аукциона;
- характеристики пользователя;
- категориальные признаки после кодирования.

Модель использует совокупность факторов, а не один доминирующий признак.

## Продакшен-готовность

- сохранён полный pipeline;
- исключена утечка данных при калибровке;
- сохранена калиброванная модель;
- обеспечена воспроизводимость предсказаний.
